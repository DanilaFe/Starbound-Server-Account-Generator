Starbound Server Account Generator

The easiest way to manage Starbound server accounts! Written in Java, this provides a simple GUI to easily modify Starbound's accesscontrol.config file, which holds the banned usernames, ips and player names as well as account names and passwords for players that can join.

Getting the app

If you want to simply download the latest file, you can get it from here. If you want to compile it yourself for whatever reason, you can do 

git clone https://github.com/DanilaFe/Starbound-Server-Account-Generator.git

and then open the project using NetBeans. (Or any IDE for that matter, the code is the same.)
